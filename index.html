GitHub Repository Készítés

Az oldal lényege, hogy megtanuljuk, hogyan kell létrehozni egy GitHub könyvtárat.
  • Az első lépés, hogy létrehozunk egy GitHub fiókot, ezt beállítjuk, testreszabjuk, ahogy szeretnénk.
  • Rákattintunk a jobbfelső sarokba, majd "Your repostories"-ra, majd szintén a jobb felsősarokban a "New"-ra.
  • Következő lépésben adunk egy nevét a "reponknak", leírást is hozzáadhatunk, a publicitását is kiválaszthatjuk, hogy mindenki lássa vagy csak saját magad, és hozzátudunk adni egy README fájlt is.
  • Majd a leírás alapján a Visual Studio Code-ban a Terminal fülön keresztül követjük amit az oldal leír.
    • Egy tutorial videó a GitHub Repokról.
